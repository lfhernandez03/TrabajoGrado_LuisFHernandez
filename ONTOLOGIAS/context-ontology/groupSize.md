# groupSize

## Metadata

- **Tipo:** Propiedad de Dato (`owl:DatatypeProperty`)
- **Ontologia:** context-ontology
- **Etiqueta:** group size

## Descripcion

Number of people watching. CRITICAL for quantitative social context: watching with a partner (2) vs. large group (5+) changes selection dynamics. Large groups need 'crowd-pleasers' with broader consensus. Can be inferred from 'there are 5 of us', 'we're 3 friends', etc.
