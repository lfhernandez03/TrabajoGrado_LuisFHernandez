# companionType

## Metadata

- **Tipo:** Propiedad de Dato (`owl:DatatypeProperty`)
- **Ontologia:** context-ontology
- **Etiqueta:** companion type

## Descripcion

Description of who the user is watching with. Typical values: 'alone', 'family', 'kids', 'partner', 'friends'. The LLM extracts this from phrases like 'I'm going to watch with my family'.
