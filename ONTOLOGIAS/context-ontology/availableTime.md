# availableTime

## Metadata

- **Tipo:** Propiedad de Dato (`owl:DatatypeProperty`)
- **Ontologia:** context-ontology
- **Etiqueta:** available time

## Descripcion

Available time in minutes. Extracted from phrases like 'I have an hour', 'something short', 'I have the whole afternoon'. The LLM converts text to a number.
