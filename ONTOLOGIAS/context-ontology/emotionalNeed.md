# emotionalNeed

## Metadata

- **Tipo:** Propiedad de Dato (`owl:DatatypeProperty`)
- **Ontologia:** context-ontology
- **Etiqueta:** emotional need

## Descripcion

What the user is seeking emotionally. Examples: 'escape stress', 'laugh', 'reflect', 'get excited'. Inferred from phrases like 'I want something that makes me laugh'.
